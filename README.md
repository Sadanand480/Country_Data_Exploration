# Study of Countries Information 🌍

An information studying task regarding the different countries of the world using Python, Pandas and NumPy.

The project involves data cleansing, data quality verification and studying to find answers to real-life problems related to demographic, economic and political data. In majority, the project pays attention to filtering, sorting, grouping and text-data analysis; visualization is a topic of another project.

## 📊 Project Elements

### Data Quality Verification

DataSets Structure & Missing Data
Statistical Summaries Data
Using .describe() method

### Exploratory Analysis

DataAnalysis of 64 demographic, economic and political parameters of 194 countries
Using filtering and sorting techniques


### Text Data

Getting information about names of the countries with the help of string manipulation
